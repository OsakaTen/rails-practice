# 社内イベント管理ポータル（Rails）

Ruby on Rails 7 を使って実装した、社内向けイベント管理ポータルです。  
社内イベントの企画・管理・公開を効率化するための、アプリとして作成しています。

> **現在の対応状況**  
> - ✅ シナリオ1：イベントの登録・編集・削除・一覧表示（管理画面）  
> - ⏳ シナリオ2：参加申し込み・確認メール送信・参加者管理（未実装）

---

## 概要

社内推進チーム(会員登録者)が主導する社内イベントを、Slack や Excel ではなく Web アプリで一元管理することを目的としたシステムです。

- 社内推進チーム(会員登録者)だけがログインしてイベントを登録・編集・削除できる
- 登録されたイベントは、公開 URL を通じて社内の誰でも閲覧できる
- 将来的には、イベントごとの参加申し込み・確認メール送信・参加者一覧の管理まで行える基盤を目指しています

---

## システム構成

- Ruby: **3.23**
- Rails: **7.2.2**
- DB: **PostgreSQL**
- テスト: **Rspec**
- 開発環境: **Dev Containers (VS Code)**

## ロール設計

| ロール             | 説明                                                                 |
|--------------------|----------------------------------------------------------------------|
| 社内推進チーム     | 管理システムにログインして、イベントの CRUD、参加者の閲覧 |
| 社員全体           | ログイン不要。イベント詳細ページの閲覧（＋今後は参加申し込み）     |

※ 現時点（v1.0.0）では、社内推進チーム向けのイベント管理（シナリオ1）のみ実装済みです。

## 実装済み機能（シナリオ1）

### 1. 管理用ログイン機能

- `メールアドレス + パスワード` による認証
- ログイン後のみ、管理画面（登録・編集・削除）にアクセス可能

※ 管理者ユーザーの作成方法（seed・サインアップの有無など）はアプリの実装に従ってください。

---

### 2. イベント管理機能（CRUD）

**管理画面から利用できる機能：**

- イベント一覧表示
- イベントの新規登録
- イベントの編集
- イベントの削除
- 公開用イベント詳細ページの URL 確認（想定）

**イベントの項目：**

| 項目         | 説明                                   |
|--------------|----------------------------------------|
| タイトル     | イベント名                             |
| 説明         | イベントの概要・目的など               |
| 開催日       | 開催日時                               |
| 主催者名     | イベント主催者（例：益田花子）         |
| 対象部署     | 全社 / 特定部署などの対象情報         |

**登録内容例：**

- タイトル：部署対抗ボードゲーム大会  
- 説明：開発部と営業部の交流を目的とした懇親イベント  
- 開催日：2024年11月15日（金）  
- 主催者名：益田花子  
- 対象部署：全社（特に開発・営業）

---

### 3. イベント詳細ページ（公開用）

- 社員であれば誰でも閲覧可能（ログイン不要）
- 管理画面で登録された内容（タイトル・説明・開催日・主催者名・対象部署）を表示
- 社内向けには、この公開 URL を Slack やメールで共有して閲覧してもらう運用を想定

## 今後実装予定（シナリオ2）

> ※ここからは **未実装** の仕様イメージです

### 参加機能（予定仕様）

**参加フォーム（イベント詳細ページ内）：**

- 氏名（必須）
- 所属部署（必須）
- メールアドレス（必須）
- 備考（任意）

**想定機能：**

- 参加申込フォームの表示（ログイン不要）
- フォーム送信後、参加者へ確認メールを自動送信
- 管理画面でイベントごとの参加者一覧を確認
- 各参加者の詳細情報（氏名・部署・メール・備考）を閲覧
